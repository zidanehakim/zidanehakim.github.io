Web3 / Modern Enhancements

  • Dot-grid hero: subtle radial-gradient dot pattern on home page background (CSS only, zero JS)
  • Glow CTAs: Resume + About buttons get box-shadow glow on hover (#7c3aed) — purple matches current accent
  • Gradient borders: project cards use conic-gradient border on hover — Web3 holographic feel
  • Skill icon tooltips: hovering a tech icon shows name in a floating pill (Radix Tooltip via shadcn)
  • Scroll progress bar: 2px violet line at top of viewport tracks scroll position within long pages (About)
  • Smooth reveal animations: sections fade+slide up as they enter viewport (Framer Motion, will-change: transform)
  • DM Sans via next/font: exact same font, zero FOUT, no render-blocking request
