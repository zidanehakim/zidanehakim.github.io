Performance Fixes (Root Cause → Fix)

🔴

WithTransition: 2000ms overflow lock

overflowY blocked for 2s on every nav
→ 300ms CSS overlay, no body lock

🔴

window.innerWidth in state

Portfolio reads window on every render
→ CSS container queries + SSR guard

🟠

Multiple raw setIntervals

3 intervals in Home typewriter
→ single useEffect, proper cleanup

🟠

Unoptimized images

Raw <img> tags, no sizing hints
→ Next.js <Image> with width/height/priority

🟠

External Google Fonts at runtime

@import url() in CSS — render-blocking
→ next/font/google (inlined, zero shift)

🟢

Lottie + heavy sections

Lazy-loaded with dynamic() + Suspense
No CLS — explicit skeletons while loading
